# Logo Dosyası

## Logo Dosyası Ekleme

Lütfen gönderilen logo görselini şu isimle kaydedin:
- **Dosya adı**: `sehitkamil-logo.png`
- **Konum**: `mobile-app/assets/images/sehitkamil-logo.png`

## Logo Özellikleri

Logo görseli şu özelliklere sahip olmalıdır:
- **Format**: PNG (şeffaf arka plan tercih edilir)
- **Çözünürlük**: Yüksek çözünürlüklü (en az 400x200px)
- **İçerik**: "Şimdi DAHA MUTLU Şehitkamil" sloganı

## Kullanım

Logo otomatik olarak şu yerlerde kullanılacaktır:
- Header component (ana sayfa ve diğer sayfalar)
- About sayfası
- Diğer gerekli yerler

Logo component'i `src/components/Logo.tsx` dosyasında tanımlıdır ve boyutlandırılabilir.
